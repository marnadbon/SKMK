# Outputs — Gegenereerde Producten

Hier komen de producten die de Product Builder genereert.

## Mappen

| Map | Inhoud |
|-----|--------|
| `cursussen/` | Complete cursusstructuren (modules, lessen, werkbladen) |
| `workshops/` | Workshop scripts en draaiboeken |
| `guides/` | Guides, PDF-structuren, werkbladen |
| `analyses/` | Materiaal-analyserapporten (`/analyseer-materiaal`) |
| `plannen/` | Productvalidatie plannen (`/productplan`) |
