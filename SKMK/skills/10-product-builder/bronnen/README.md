# Bronnen — Bronmateriaal voor Phase B

Zet hier bronmateriaal neer waaruit de Product Builder methodologie kan extraheren.

## Aanbevolen bronmateriaal

### Cursus creatie methodologie
- Hoe je een online cursus curriculum structureert
- Learning design / instructional design voor online cursussen
- "Hoe ik mijn cursus heb gebouwd" breakdowns

### Workshop facilitatie
- Hoe je een engaging live workshop geeft
- Workshop design en energiemanagement
- Webinar structuur die converteert

### Product validatie
- Hoe je een infoproduct valideert voor je bouwt
- Pre-selling methodologie
- MVP-aanpak voor cursussen/workshops

### Materiaalverwerking
- Hoe je expertise omzet naar een cursus
- Curriculum design from scratch
- De "vloek van kennis" en hoe je beginners leert

## Formaat

- YouTube transcripties als `.md` of `.txt`
- Beschrijvende bestandsnamen (bijv. `youtube-how-i-built-my-course.md`)
