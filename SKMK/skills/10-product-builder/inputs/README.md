# Inputs — Ruwe Materialen

Zet hier de ruwe materialen neer waaruit de Product Builder je product gaat structureren.

## Wat je hier kunt neerzetten

- **Transcripties** — van video's, podcasts, of webinars (.md of .txt)
- **Notities** — braindumps, outlines, steekwoorden
- **Bestaande content** — blogposts, artikelen, of presentaties
- **Onderzoek** — marktonderzoek, klantinterviews, survey resultaten
- **Afbeeldingen** — screenshots, diagrammen, slides

## Formaat

- Markdown (.md) bestanden hebben de voorkeur
- Naam bestanden beschrijvend (bijv. `webinar-transcriptie-aanbod.md`)
- Hoe meer context je meegeeft, hoe beter het resultaat
