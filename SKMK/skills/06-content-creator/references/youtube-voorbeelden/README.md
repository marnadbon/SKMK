# YouTube Voorbeelden

Voeg hier YouTube scripts toe die goed presteren. De Content Creator gebruikt deze als referentie bij het schrijven van scripts.

## Hoe te gebruiken

- Plak scripts als `.md` bestanden (bijv. `script-tutorial-seo.md`, `script-story-driven-reis.md`)
- Of plak getranscribeerde video's (eigen of van anderen)
- Voeg context toe: wie maakte het, hoeveel views/retentie, waarom het werkte

## Wat werkt als referentie

- Eigen scripts die goed scoorden (hoge retentie, veel views)
- Getranscribeerde video's van creators die je bewondert
- Scripts die specifiek het type resultaat genereerden dat je wilt (subscribers, leads, views)
- Video's met opvallend goede hooks of storytelling

De agent leest deze map bij het schrijven van YouTube scripts via `/youtube-script`.
