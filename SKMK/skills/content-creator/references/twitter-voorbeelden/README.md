# X/Twitter Voorbeelden

Voeg hier tweets en threads toe die goed presteren. De Content Creator gebruikt deze als referentie bij het schrijven van X/Twitter content.

## Hoe te gebruiken

- Plak tweets als `.md` bestanden (bijv. `tweet-hot-take-ai.md`, `thread-framework-productiviteit.md`)
- Of maak één bestand met meerdere tweets gescheiden door `---`
- Voeg context toe: wie schreef het, hoeveel engagement het kreeg, waarom het werkte

## Wat werkt als referentie

- Eigen tweets die goed presteerden (hoge impressies, veel replies)
- Threads die veel engagement genereerden
- Tweets van anderen in je niche die je bewondert
- Tweets die specifiek het type engagement genereerden dat je wilt (replies, retweets, DM's)

De agent leest deze map bij het schrijven van X/Twitter content via `/social-post twitter`.
