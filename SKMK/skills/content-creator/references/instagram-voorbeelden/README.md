# Instagram Voorbeelden

Voeg hier Instagram Reels scripts, captions en carousel concepten toe die goed presteren. De Content Creator gebruikt deze als referentie bij het maken van Instagram content.

## Hoe te gebruiken

- Plak content als `.md` bestanden (bijv. `reel-script-tips.md`, `caption-behind-the-scenes.md`)
- Of maak één bestand met meerdere voorbeelden gescheiden door `---`
- Voeg context toe: wie maakte het, hoeveel engagement het kreeg, waarom het werkte

## Wat werkt als referentie

- Eigen Reels die goed presteerden (hoge views, saves, DM's)
- Captions die veel comments genereerden
- Carousel posts met hoge saves en shares
- Content van anderen in je niche die je bewondert
- Posts die specifiek het type engagement genereerden dat je wilt (DM's, saves, follows)

De agent leest deze map bij het schrijven van Instagram content via `/social-post instagram`.
