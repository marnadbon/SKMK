# References — Voorbeeldcontent

Voeg hier je best presterende content toe zodat de agent je stijl en aanpak leert.

## Platform Mappen

| Map | Wanneer gebruikt |
|-----|-----------------|
| `linkedin-voorbeelden/` | Bij LinkedIn posts via `/social-post linkedin` |
| `twitter-voorbeelden/` | Bij X/Twitter posts via `/social-post twitter` |
| `youtube-voorbeelden/` | Bij YouTube scripts via `/youtube-script` |
| `instagram-voorbeelden/` | Bij Instagram content via `/social-post instagram` |

## Formaat

- Markdown (.md) bestanden
- Naam beschrijvend (bijv. `linkedin-post-over-ai.md`, `youtube-script-productiviteit.md`)
- Voeg bovenaan een korte notitie toe over waarom dit goed presteerde

## Hoe de agent dit gebruikt

De agent leest bestanden in de relevante platform map om je contentpatronen te leren. Meer voorbeelden = betere output. Streef naar minimaal 5-10 referentiebestanden per platform.
