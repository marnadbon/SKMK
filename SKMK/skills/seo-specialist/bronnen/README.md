# Bronnen — SEO Bronmateriaal

Plaats hier je eigen SEO-bronmateriaal: transcripties, notities, cursusmateriaal, of guidelines.

## Wat hier hoort

- Transcripties van SEO-lessen of workshops
- Eigen notities over je SEO-aanpak
- Guidelines voor landingspagina's
- Voorbeelden van goed presterende pagina's (met context)

## Formaat

- Markdown (`.md`) of plain text (`.txt`)
- Geef bestanden beschrijvende namen (bijv. `seo-pagina-guidelines-cursus.md`)
- Voeg bovenaan een korte notitie toe over de bron

## Hoe de agent dit gebruikt

In **Phase B** leest de agent deze bronnen en extraheert:
- Jouw specifieke SEO-methodologie
- Pagina-templates op basis van jouw richtlijnen
- Technische SEO-regels die je volgt
- Interne linking-strategieen die je toepast

De geextraheerde kennis vult de bestanden in `knowledge-base/` — gemarkeerd met `<!-- BRON -->` placeholders.
