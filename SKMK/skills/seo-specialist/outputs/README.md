# Outputs — Gegenereerde SEO Content

Hier slaat de agent gegenereerde content op:

- **Landingspagina's** uit `/genereer-paginas`
- **Keyword maps** uit `/keyword-map`
- **SEO audit rapporten** uit `/seo-audit`

## Naamgeving

- `paginas/[patroon]-[datum].md` — gegenereerde pagina-batches
- `keyword-maps/[niche]-[datum].md` — keyword maps
- `audits/[onderwerp]-[datum].md` — audit rapporten

De agent maakt submappen automatisch aan wanneer nodig.
