# References — Voorbeeldproducten

Zet hier voorbeelden van goed opgebouwde producten neer als referentie.

## Wat je hier kunt neerzetten

- **Cursusstructuren** — outlines van cursussen die je goed vindt
- **Workshop scripts** — voorbeelden van effectieve workshops
- **Guide templates** — goed opgezette guides of PDF's
- **Concurrentie** — productstructuren van concurrenten (ter analyse)

## Formaat

- Markdown (.md) bestanden hebben de voorkeur
- Naam beschrijvend (bijv. `cursus-voorbeeld-ali-abdaal.md`)
