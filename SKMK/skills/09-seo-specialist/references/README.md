# References — Voorbeeldpagina's

Voeg hier voorbeelden toe van goed presterende landingspagina's en SEO-content.

## Wat hier hoort

- Eigen landingspagina's die goed ranken
- Concurrenten-pagina's die je wilt overtreffen
- Voorbeelden van goede programmatische SEO (bijv. Wise, Zapier, HubSpot)
- Screenshots of exports van pagina's met ranking-data

## Formaat

- Markdown (`.md`) bestanden
- Naam beschrijvend (bijv. `wise-valuta-paginas.md`, `eigen-dienst-amsterdam.md`)
- Voeg bovenaan een korte notitie toe:
  - Waarom dit een goed voorbeeld is
  - Welk zoekwoord/patroon het target
  - Ranking-positie (als bekend)

## Hoe de agent dit gebruikt

De agent leest deze map bij het genereren van pagina's via `/genereer-paginas` en bij het schrijven van SEO-audits via `/seo-audit`. Goede voorbeelden helpen de agent om de kwaliteitslat hoger te leggen.
